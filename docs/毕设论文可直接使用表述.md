# 毕设论文可直接使用表述

## 1. 项目定位

本系统定位为基于 Python 与 YOLOv8 的智能垃圾多目标检测与回收处理系统，能够对图片、视频和摄像头画面中的多个垃圾目标进行识别，并在识别结果基础上给出垃圾类别归属、回收建议和知识说明。

## 2. 数据集构建说明

针对原项目中存在的整图默认框问题，本文重新构建了面向目标检测的数据集。数据集采用 YOLO 标注格式，对图像中的每一个主要垃圾目标进行独立边界框标注，并对不清晰、类别难以判定或不适合检测的样本进行筛除。考虑到纸箱与纸类在回收属性和材质特征上的接近性，本文将原 cardboard 类合并至 paper 类，最终形成9类垃圾检测数据集。

## 3. 模型训练说明

考虑到实验设备以 GTX 1650 为主，本文采用 YOLOv8n 作为主训练模型，并设计了两阶段训练策略。第一阶段冻结主干网络的部分层，仅训练检测头，使模型快速适配垃圾类别特征；第二阶段解冻全部网络参数进行微调，从而提升整体检测精度和边界框定位能力。在此基础上，额外训练了 YOLOv8s 版本作为精度增强对比模型。

## 4. 视频识别优化说明

为兼顾视频识别的流畅度与准确性，本文并未对每一帧都执行完整目标检测，而是采用间隔帧检测策略，并在连续帧之间基于 IoU 实现目标关联。针对视频中检测框抖动和类别跳变的问题，本文进一步加入边界框平滑与类别投票机制，从而提升边界框的稳定性与目标跟随度。

## 5. 系统功能说明

系统前端支持图片识别、视频识别、摄像头实时识别、垃圾分类知识展示、历史记录查询和回收点检索。后端采用 Node.js 作为统一业务网关，负责文件接收、结果封装与历史记录管理；Python 服务负责模型推理与视频后处理，从而实现识别功能与回收处理建议功能的闭环集成。
